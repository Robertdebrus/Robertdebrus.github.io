EyeDoc

Prerequisites:

Firefox Bash on Ubuntu on Windows

1. Installation

Install the EyeTribe SDK from .zip below.
Install cherrypy in Bash
Install CORS Everywhere add-on for Firefox (Here)

2. Setup

Run EyeTribeUI and follow setup instructions.
Calibrate EyeTracker (following demo)
Run Bash on Ubuntu on Windows, navigate to eyeDoc/python/, run: python tcpmiddleware.py
open eyeDoc/apidocs/index.html in Firefox

3. Usage


Wink or Blink to click
If you are having trouble with wink or blink clicking, try zooming the browser in more.
